#  Bank Customer Churn Analysis

##  Overview
This project analyzes bank customer data to understand patterns of customer churn and predict which customers are likely to leave the bank using data science techniques.

---

##  Objective
- Identify customers who are likely to churn  
- Analyze factors influencing customer retention  
- Build a predictive model for churn  

---

##  Concepts Used
- Data Preprocessing  
- Data Transformation  
- Descriptive Statistics  
- Exploratory Data Analysis (EDA)  
- Linear Regression  
- Correlation Analysis  

---

##  Tools & Technologies
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

##  Dataset
- Bank Customer Churn Dataset  
- ~10,000 customer records  
- Includes demographic, financial, and behavioral data

##  Model Output
- Linear Regression model used for prediction  
- Predictions converted into binary classification  
- Achieved **~75% accuracy**  

---

##  Key Insights
- Most customers fall in the **30–40 age range**  
- Age distribution shows slight **right skew due to older customers**  
- Churn behavior varies across **age groups**  
- Customers with **lower engagement** are more likely to churn  
- Balance influences customer retention
